Installation
---------------

.. start installation

.. tabs::

	.. tab:: from PyPI

		.. prompt:: bash

			python3 -m pip install wxIconSaver --user


	.. tab:: from GitHub

		.. prompt:: bash

			python3 -m pip install git+https://github.com/domdfcoding/wxIconSaver@master --user

.. end installation

``wxIconSaver`` can then be run with the ``wxIconSaver`` command.
